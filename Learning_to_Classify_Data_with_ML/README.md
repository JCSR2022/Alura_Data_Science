# Customer Investment Adherence Prediction üìà

This repository contains a Machine Learning project focused on **Supervised Classification**. The goal is to predict whether a bank's clients will adhere to a specific investment product based on historical marketing campaign data.

By utilizing predictive modeling, companies can optimize their sales strategies, improve decision-making, and personalize product recommendations.

---

## üéØ Project Overview

The core of this project is to build a classification pipeline that identifies potential investors. This allows the marketing team to focus their efforts on high-probability leads, thereby increasing the conversion rate and ROI of the campaign.

### Topics Covered:

* **Exploratory Data Analysis (EDA):** Understanding data distributions and patterns.
* **Feature Engineering:** Transforming categorical variables for model compatibility.
* **Model Training:** Implementing and tuning classification algorithms.
* **Inference:** Making predictions on unseen data.
* **Persistence:** Saving transformation pipelines and models using `pickle`.
* **Business Impact:** Applying ML to customer segmentation and growth strategies.
